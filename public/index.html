This project is a demonstration of setting up Continuous Integration and Continuous Deployment (CI/CD) using Jenkins and GitHub.

📌 About This Application

This is a basic Node.js Express application designed to showcase the DevOps pipeline workflow.

It includes a simple server that renders this HTML page as a response to HTTP requests.

🧰 Technologies Used

  • Node.js (Express Framework)
  • Jenkins for CI/CD Pipeline
  • GitHub for version control and repository management
  • dotenv for environment variable configuration

🖥️ Server Status

The server is up and running. You are viewing this page served directly through Express from the public directory.

🎯 Objective

The goal of this assignment is to gain practical experience in DevOps by building and deploying a real application using industry-standard tools and practices.
